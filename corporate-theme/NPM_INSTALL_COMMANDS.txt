DANH SÁCH TẤT CẢ LỆNH NPM ĐỂ CÀI THƯ VIỆN TRONG DỰ ÁN
=====================================================

CÁCH 1: CÀI TẤT CẢ CÙNG LÚC (KHUYẾN NGHỊ)
==========================================
npm install

Lệnh này sẽ tự động cài tất cả dependencies và devDependencies từ package.json


CÁCH 2: CÀI TỪNG THƯ VIỆN RIÊNG LẺ
===================================

A. DEPENDENCIES (Production - Thư viện chạy trong production)
--------------------------------------------------------------

1. Font Awesome Icons:
   npm install @fortawesome/fontawesome-free

2. Bootstrap CSS Framework:
   npm install bootstrap

3. GSAP (GreenSock Animation Platform):
   npm install gsap

4. Swiper (Carousel/Slider):
   npm install swiper

5. AOS (Animate On Scroll):
   npm install aos

6. jQuery:
   npm install jquery

7. Animate.css:
   npm install animate.css

8. Isotope Layout:
   npm install isotope-layout

9. jQuery Ripples (Water ripple effect):
   npm install jquery.ripples

10. Magnific Popup (Lightbox):
    npm install magnific-popup

11. Particles.js:
    npm install particles.js

12. Smooth Scrollbar:
    npm install smooth-scrollbar

13. Waypoints:
    npm install waypoints

14. WOW.js (Scroll animations):
    npm install wow.js

15. TypeScript Node Types:
    npm install @types/node


B. DEV DEPENDENCIES (Development - Chỉ dùng khi phát triển)
-----------------------------------------------------------

1. TypeScript:
   npm install --save-dev typescript

2. Vite (Build tool):
   npm install --save-dev vite

3. SASS (CSS Preprocessor):
   npm install --save-dev sass

4. TypeScript Definitions cho AOS:
   npm install --save-dev @types/aos

5. TypeScript Definitions cho Isotope:
   npm install --save-dev @types/isotope-layout

6. TypeScript Definitions cho jQuery:
   npm install --save-dev @types/jquery

# Chỉ cần chạy 1 lệnh này (nếu đã có package.json):
npm install


LƯU Ý
=====
- Nếu đã có package.json và package-lock.json, chỉ cần: npm install
- Nếu cài từng thư viện, npm sẽ tự động cập nhật package.json
- Dùng --save-dev cho devDependencies
- Dùng --save (hoặc không có flag) cho dependencies
- Sau khi cài, kiểm tra node_modules/ đã có thư viện chưa
